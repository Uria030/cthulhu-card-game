## 一、通用規則

### 1.1 費用與價值總則

> **詳細的價值計算公式、效果價值表、各類修正值請參照《卡片價值計算規範 v0.1》。**

| 原則 | 說明 |
|------|------|
| 費用範圍 | 0–6 資源 |
| 費用計算 | 依卡片類型套用對應公式（事件卡 1:1、資產卡 2:1、盟友卡 1:1） |
| 等級抵扣 | 0 級 = 0，1 級 = -1V，2 級 = -2V ... 5 級 = -5V |
| 稀有度抵扣 | 隨身 = 0，基礎 = -1V，標準 = -2V，進階 = -3V，稀有 = -4V，傳奇 = -5V |
| 條件限制 | 不影響價值計算 |
| 負面副作用 | 直接扣除對應效果價值作為代價 |

### 1.2 卡片等級

| 等級 | 說明 |
|------|------|
| 0 級 | 基礎卡，起始可用 |
| 1 級 | 成長後獲取 |
| 2 級 | 成長後獲取 |
| 3 級 | 成長後獲取 |
| 4 級 | 成長後獲取 |
| 5 級 | 成長後獲取 |

- 任何等級的卡片費用都可以是 0-6
- 升級時卡片可能獲得能力提升（價值增加），最終費用不一定變便宜
- 同一張卡從 0 級可升級到 5 級

### 1.3 稀有度

| 稀有度 | 說明 | 價值抵扣 |
|--------|------|----------|
| 隨身 | 最基礎的工具 | 0 |
| 基礎 | 基礎工具和能力 | -1V |
| 標準 | 有特色的裝備和能力 | -2V |
| 進階 | 強力且有主題性的卡片 | -3V |
| 稀有 | 稀有強力卡片 | -4V |
| 傳奇 | 改變遊戲規則的卡片 | -5V |

稀有度有雙重作用：
1. **抵扣價值** → 降低費用
2. **允許更強效果** → 決定能力價值上限

### 1.4 消耗類型修正（暫定）

| 消耗類型 | 修正 | 說明 |
|----------|------|------|
| 留在場上（stay） | -1V | 打出當下無即時價值 |
| 進入棄牌堆（discard） | 0 | 基準，洗牌後可再抽到 |
| 短休息回復（short_rest） | -1V | 比 discard 受限 |
| 長休息回復（long_rest） | -2V | 比短休息更受限 |
| 移除出遊戲（removed） | -3V | 最嚴格，只能用一次 |

消耗限制用於平衡過強效果：效果越強 → 使用限制越嚴格 → 價值修正越多 → 費用平衡

## 二、物理武器卡（Weapon Cards）

### 2.1 傷害階層

| 稀有度 | 傷害 | 彈藥 | 說明 |
|--------|------|------|------|
| 隨身 | 1 | 無限 | 小刀、拳頭，無消耗 |
| 基礎 | 2 | 3 | .32 左輪手槍 |
| 標準 | 3 | 3 | .45 自動手槍 |
| 進階 | 4 | 2 | 獵槍 |
| 稀有 | 5 | 2 | 魔法武器 |
| 傳奇 | 6 | 2 | 散彈槍 |

> **武器彈藥打完後進棄牌堆**，需再次打出才能繼續使用。

### 2.2 武器費用公式

> **詳見《卡片價值計算規範 v0.1》第三章：資產卡費用公式**

```
費用 = （總輸出價值 - 留場修正 - 消耗類型修正 - 等級抵扣 - 稀有度抵扣）÷ 2
```

**計算範例：散彈槍**

| 項目 | 數值 | 說明 |
|------|------|------|
| 傷害 × 彈藥 | 18V | 6 × 3 |
| 留場修正 | -1V | 資產類 |
| 消耗類型 | 0 | 子彈打完進棄牌堆 |
| 等級抵扣 | -4V | 4 級 |
| 稀有度抵扣 | -5V | 傳奇 |
| **計算總價值** | **8V** | |
| 資產換算比例 | 2:1 | |
| **費用** | **4** | |

### 2.3 武器欄位規格

| 欄位 | 規格 | 說明 |
|------|------|------|
| `type` | `asset` | 武器是資產卡 |
| `subtypes` | 必須包含 `weapon` + 具體類型 | `weapon_melee` / `weapon_ranged` |
| `slot` | `one_hand` 或 `two_hand` | 傷害 1–3 通常單手，4–6 通常雙手 |
| `combat_style` | 必填 | 8 種戰鬥風格之一 |
| `attribute_modifiers` | JSONB | 基礎武器只有 1 個屬性修正 |
| `ammo` | 彈藥數 | 遠程武器必填，近戰武器為 NULL |
| `consume_type` | `discard` | 彈藥打完進棄牌堆 |

### 2.4 彈藥規格

| 武器類型 | 彈藥數 | 設計意圖 |
|---------|--------|---------|
| 近戰武器 | 無 | 不消耗，但需要交戰 |
| 手槍 | 3 | 標準續航 |
| 步槍 / 獵槍 | 2–3 | 高傷害低續航 |
| 散彈槍 | 2 | 最高傷害最低續航 |
| 弓弩 | 3–4 | 標準續航 |

### 2.5 屬性修正規格

| 武器等級 | 修正結構 | 範例 |
|---------|---------|------|
| 0 級（基礎） | 單一屬性 +1 | `{ "strength": 1 }` |
| 1–2 級 | 單一屬性 +1 至 +2 | `{ "agility": 2 }` |
| 3 級 | 單一 +2 或雙屬性各 +1 | `{ "strength": 1, "agility": 1 }` |
| 4–5 級 | 可出現 `"all": 1` 或 單一 +3 | `{ "all": 1 }` 或 `{ "strength": 3 }` |

> 負面修正不出現在基礎卡上，只在玩家強化路線中作為取捨。

### 2.6 武器與戰鬥專精的關係

每張武器卡透過 `combat_style` 歸屬一個第一層戰鬥風格（8 種之一）。

| 層級 | 對武器的影響 | 範例 |
|------|------------|------|
| 無熟練 | 可以使用武器，但攻擊時無熟練修正 | 學者拿起獵槍，+0 熟練 |
| 第一層（熟練） | 攻擊時加上熟練修正值 | 有槍枝射擊熟練的偵探，+1 熟練 |
| 第二層（專精） | 更高修正值 + 專精特有效果 | 手槍專精的殺手，+2 熟練 + 快速拔槍 |

**八種戰鬥風格與武器類型的對應：**

| combat_style | 典型武器 | 第二層專精方向 |
|-------------|---------|--------------|
| `shooting` | 手槍、步槍、衝鋒槍、散彈槍 | 步槍、衝鋒槍、雙槍、手槍 |
| `archery` | 弓、弩、投擲武器 | 獵手、連射、毒箭、無聲射手 |
| `sidearm` | 匕首、棍棒、鐵管、酒瓶 | 匕首術、護身格擋、鈍擊、街頭格鬥 |
| `military` | 大劍、戰斧、長槍、盾牌 | 雙手武器、防禦架式、雙持、長柄武器 |
| `brawl` | 拳套、指虎、無武器 | 酒館鬥毆者、摔角大師、空手道 |
| `arcane` | 法杖、法器、施法媒介 | 儀式、咒語、引導、冥想、煉金 |
| `engineer` | 炸藥、陷阱、機關裝置 | 爆破、陷阱、機械 |
| `assassin` | 暗器、毒針、細線 | 無聲處決、伏擊戰術、暗器 |

### 2.7 弓弩箭矢系統

弓弩類武器（`combat_style: archery`）有獨特的**箭矢裝備機制**：

| 項目 | 規格 |
|------|------|
| 基礎彈藥 | 弓弩自帶的箭矢為**物理元素** |
| 特殊箭矢 | 可透過消耗品卡片更換箭矢類型 |
| 裝填方式 | 打出箭矢消耗品卡 → 裝填到弓弩上 → 接下來的 X 發攻擊使用該箭矢 |
| 元素覆蓋 | 特殊箭矢**覆蓋**弓弩的基礎物理元素 |

**箭矢消耗品規格：**

| 箭矢 | 費用 | 發數 | 元素 | 附加效果 | consume_type |
|------|------|------|------|---------|-------------|
| 火箭 | 1 | 3 | 火 | 命中施加燃燒 1 | removed |
| 冰箭 | 1 | 3 | 冰 | 命中施加冷凍 1 | removed |
| 電擊箭 | 2 | 3 | 電 | 對潮濕目標額外 +2 傷害 | removed |
| 毒箭 | 1 | 3 | 物理 | 命中施加中毒 1 | removed |
| 銀箭 | 2 | 2 | 神秘 | 穿透物理抗性 | removed |

### 2.8 特殊彈藥通用規格

| 彈藥 | 費用 | 發數 | 元素 | 適用武器 | consume_type |
|------|------|------|------|---------|-------------|
| 銀彈 | 2 | 3 | 神秘 | 所有槍械 | removed |
| 燃燒彈 | 2 | 2 | 火 | 散彈槍 | removed |
| 穿甲彈 | 1 | 3 | 物理 | 所有槍械，無視物理抗性 1 點 | removed |

## 三、法術卡（Spell Cards）

### 3.1 核心設計原則

- **法術一定命中**，混沌袋決定副作用（見補充 04）
- 法術的傷害元素**五種都有可能**（物理/火/冰/電/神秘），由法術本身的主題決定
- **神秘元素是遊戲最強元素**（無任何怪物有抗性），但不代表所有法術都是神秘元素
- 法術比同級物理武器**傷害略低**，但命中率 100% 彌補了這個差距
- 法術的真正代價不在費用，在混沌袋的副作用風險

### 3.2 法術的兩個分類維度

**法術類型（spell_type）— 法術做什麼：**

| 代碼 | 中文 | 核心功能 |
|------|------|---------|
| `combat` | 戰鬥與毀滅 | 造成傷害 |
| `investigation` | 調查與預言 | 獲取資訊、發現線索 |
| `protection` | 防護與迴避 | 保護、閃避 |
| `spacetime` | 時空與位面操控 | 封印、傳送、空間操控 |
| `summoning` | 召喚與束縛 | 召喚援軍、控制敵人 |
| `healing` | 治癒與淨化 | 恢復、淨化 |

**施法方式（spell_casting）— 法術怎麼施放：**

| 代碼 | 中文 | 對卡片規格的影響 |
|------|------|----------------|
| `ritual` | 儀式 | 費用較高、充能較少、效果最強 |
| `incantation` | 咒語 | 即時施放、標準費用、標準充能 |
| `channeling` | 引導 | 持續型效果、需要維持（橫置） |
| `meditation` | 冥想 | 偏向恢復和防禦、費用較低 |
| `alchemy` | 煉金 | 產出消耗品形式的法術效果 |

### 3.3 法術傷害階層

| 階層 | 傷害 | 費用 | 充能 | 對比物理武器 |
|------|------|------|------|------------|
| 初階 | 1 | 1 | 3–4 | 隨身武器（傷害 1），但法術一定命中 |
| 基礎 | 2 | 2 | 3 | 基礎武器（傷害 2），法術無需彈藥但有副作用 |
| 標準 | 3 | 3 | 3 | 標準武器（傷害 3），等傷害但法術穿透抗性 |
| 進階 | 4 | 4 | 2–3 | 進階武器（傷害 4），充能較少 |
| 稀有 | 5 | 5 | 2 | 稀有武器（傷害 5），充能最少 |
| 傳奇 | 6 | 6 | 1–2 | 傳奇武器（傷害 6），極稀有 |

### 3.4 法術卡專用欄位

| 欄位 | 規格 | 說明 |
|------|------|------|
| `subtypes` | 必須包含 `weapon_arcane` 或 `arcane_item` | 戰鬥型法術用 `weapon_arcane` |
| `spell_type` | 必填 | 6 種法術類型之一 |
| `spell_casting` | 必填 | 5 種施法方式之一 |
| `combat_style` | `arcane` | 戰鬥型法術固定為施法風格 |
| `slot` | `arcane` | 法術佔據神秘欄位 |
| `uses` | 充能數 | 資產型法術必填 |

## 四、一般道具卡（Item Cards）

### 4.1 分類

| 子類型 | 說明 | 典型卡片類型 |
|--------|------|------------|
| `item` | 一般工具 | 資產 |
| `consumable` | 消耗品 | 事件或資產（用完棄掉/移除） |
| `light_source` | 光源 | 資產 |

### 4.2 一般工具規格

| 項目 | 規格 |
|------|------|
| 費用 | 0–3（工具通常便宜） |
| 效果類型 | 檢定加值、資源產生、卡牌操控、環境互動 |
| 檢定加值範圍 | +1 至 +2（單一屬性） |
| 欄位 | `accessory`、`head`、`body`、`none` |
| `consume_type` | `stay`（留在場上） |

### 4.3 消耗品規格

| 項目 | 規格 |
|------|------|
| 費用 | 0–3（比同效果的永久道具便宜） |
| 效果強度 | 比同費用的資產卡高（因為用完就沒了） |
| `consume_type` | `discard`、`short_rest`、`long_rest`、`removed` |

### 4.4 光源規格

| 項目 | 規格 |
|------|------|
| 費用 | 0–2 |
| 照明範圍 | 所在板塊（基礎）、所在 + 相鄰板塊（進階） |
| 使用次數 | 3–6 次 |
| 欄位 | `accessory` 或 `none` |

| `consume_type` | `stay`（留場）或 `removed`（使用完移除） |

## 六、盟友卡（Ally Cards）

### 6.1 盟友定位

盟友是可以替調查員擋傷害、提供被動增益、甚至參與戰鬥的 NPC 夥伴。

### 6.2 盟友費用公式

> **詳見《卡片價值計算規範 v0.1》第三章：盟友卡費用公式**

```
費用 = 總價值 - 等級抵扣 - 稀有度抵扣
```

**總價值 = HP 價值 + SAN 價值 + 留場價值 + 能力價值**

| 項目 | 價值 |
|------|------|
| 盟友 1 HP | 0.5V |
| 盟友 1 SAN | 0.5V |
| 留場價值 | 2V |
| 持續被動（單屬性 +1） | 0.5V |
| 有代價的能力 | 0V |

### 6.3 盟友數值規格

| 項目 | 規格 |
|------|------|
| 卡片類型 | `ally` |
| 費用 | 2–5 |
| 欄位 | 無（盟友不佔裝備欄位） |
| 場上限制 | 每位調查員同時最多 **1 位盟友** |
| `consume_type` | `stay`（留在場上直到被摧毀） |

### 6.4 盟友 HP/SAN 規格

| 盟友定位 | HP | SAN | 費用 | 設計意圖 |
|---------|-----|------|------|---------|
| 肉盾型 | 3–4 | 1 | 2–3 | 大量承受物理傷害，但精神脆弱 |
| 精神支柱型 | 1 | 3–4 | 2–3 | 大量承受恐懼傷害，但肉體脆弱 |
| 平衡型 | 2 | 2 | 2–3 | 兩方面都能擋一些 |
| 戰鬥型 | 2–3 | 1–2 | 3–4 | HP 適中，主要價值是戰鬥能力 |
| 工具型 | 1 | 1 | 2–4 | HP/SAN 低，主要價值是特殊能力 |

### 6.5 盟友能力類型

| 類型 | 說明 | 範例 |
|------|------|------|
| 被動增益 | 在場期間持續提供加成 | 「你的調查檢定 +1」 |
| 反應能力 | 特定條件觸發 | 「你受到傷害時，可以將 1 點傷害分配到此盟友」 |
| 主動能力 | 花費行動點或橫置觸發 | 「橫置此盟友：攻擊 1 個交戰中的敵人，造成 2 點傷害」 |
| 犧牲能力 | 摧毀盟友觸發強力一次性效果 | 「摧毀此盟友：自動成功一次檢定」 |
| 進場效果 | 打出時觸發一次 | 「進場時：恢復 2 點 SAN」 |
| 離場效果 | 被摧毀時觸發 | 「被摧毀時：對交戰中的敵人造成 3 點傷害」 |

### 6.6 盟友戰鬥規格

| 項目 | 規格 |
|------|------|
| 盟友攻擊力 | 1–3 傷害（不超過同費用武器的傷害） |
| 攻擊方式 | 橫置此盟友（每回合 1 次）或消耗使用次數 |
| 檢定 | 盟友攻擊**不需要調查員擲骰**，自動命中 |
| 元素 | 物理（普通盟友）或神秘（超自然盟友，此時觸發混沌袋） |

## 七、技能卡（Skill Cards）

### 7.1 定位

技能卡是專門用於**檢定加值**的卡片。打出時不佔行動點，在檢定前從手牌投入，使用後進棄牌堆。

### 7.2 技能卡規格

| 項目 | 規格 |
|------|------|
| 卡片類型 | `skill` |
| 費用 | **0**（技能卡不花資源，代價是佔手牌空間） |
| 加值範圍 | +1 至 +3（對應的屬性） |
| 加值屬性 | 可以是 1–2 個屬性 |
| 使用方式 | 檢定前從手牌投入，自動進棄牌堆 |
| 附加效果 | 可以有「檢定成功/失敗時」的額外效果 |

### 7.3 技能卡等級價值區間

> **詳見《卡片價值計算規範 v0.1》第五章：技能卡等級價值區間**

| 等級 | 價值區間 | 說明 |
|------|----------|------|
| LV0 | 2.5V - 3V | 基礎效果 |
| LV1 | 2.5V - 3V | +功能彈性 |
| LV2 | 5V - 6V | |
| LV3 | 7V - 8V | |
| LV4 | 8V - 9V | 推測 |
| LV5 | 9V - 10V | |

## 八、事件卡通用規格（Event Cards）

### 8.1 定位

事件卡是一次性效果，打出後進棄牌堆。

### 8.2 事件卡費用公式

> **詳見《卡片價值計算規範 v0.1》第三章：事件卡費用公式**

```
費用 = 總價值 - 等級抵扣 - 稀有度抵扣 - 負面副作用
```

換算比例 = **1:1**

## 九、跨類別平衡錨點

### 9.1 傷害效率對照

| 來源 | 每行動點期望傷害 | 條件 |
|------|---------------|------|
| 隨身武器 | 0.6–0.8（命中率 × 傷害 1） | 需交戰，無消耗 |
| 標準武器 | 1.5–2.0（命中率 × 傷害 3） | 消耗彈藥 |
| 標準法術 | 3.0（一定命中 × 傷害 3） | 消耗充能 + 混沌袋風險 |
| 盟友攻擊 | 1.0–2.0（自動命中 × 傷害 1–2） | 需橫置盟友 |
| 消耗品（火焰瓶） | 4.0+（範圍 2 傷害 × 多目標） | 永久消耗 |

### 9.2 恢復效率對照

| 來源 | 每資源恢復量 | 備註 |
|------|------------|------|
| 急救繃帶（費用 0） | 1 HP / 0 資源 | 一次性，佔手牌 |
| 急救箱（費用 2） | 6 HP / 2 資源（3 次 × 2 HP） | 資產，可回收 |
| 治癒法術（費用 2） | 6–9 HP / 2 資源 | 有混沌袋風險 |
| 威士忌（費用 1） | 2 SAN / 1 資源（但 -1 HP） | 有代價的恢復 |

## 十、設計檢查清單

設計每張卡片時，逐項確認：

**通用：**
- [ ] 費用是否落在 0–6 範圍內？
- [ ] 是否依照《卡片價值計算規範》計算費用？
- [ ] 等級抵扣是否正確應用？
- [ ] 稀有度抵扣是否正確應用？
- [ ] 陣營歸屬是否與該陣營的機制關鍵字吻合？
- [ ] 效果語言是否使用補充 02 定義的合法代碼？

**武器卡：**
- [ ] 傷害值是否符合對應稀有度階層？
- [ ] 彈藥數是否在合理範圍（參照 2.4）？
- [ ] 是否標註了 `combat_style`？
- [ ] 是否標註了 `attribute_modifiers`？
- [ ] 是否使用資產卡 2:1 換算比例？

**法術卡：**
- [ ] 是否標註了 `spell_type` 和 `spell_casting`？
- [ ] 元素屬性是否與法術主題匹配？
- [ ] 充能數是否低於同級物理武器的彈藥數？

**盟友卡：**
- [ ] HP + SAN 總和是否在合理範圍？
- [ ] 是否正確計算 HP/SAN/留場/能力價值？
- [ ] 有代價的能力是否計為 0V？
- [ ] 是否使用盟友卡 1:1 換算比例？

**事件卡：**
- [ ] 是否使用事件卡 1:1 換算比例？
- [ ] 負面副作用是否正確扣除價值？

**技能卡：**
- [ ] 總價值是否落在對應等級的價值區間？
- [ ] 費用是否為 0？

>
> 本文件是卡片設計的規格權威，《卡片價值計算規範》是數值計算的權威。
> 設計任何卡片前，先對照本文件的對應章節，並使用《卡片價值計算規範》計算費用。
>
> 特別注意：
> 1. **費用計算必須套用公式** — 不要憑感覺定價，使用《卡片價值計算規範》的公式。
> 2. **等級與費用無固定關係** — 任何等級都可以是 0-6 費，等級只影響抵扣。
> 3. **稀有度有雙重作用** — 既抵扣價值，也決定能力強度上限。
> 4. **資產卡 2:1、事件/盟友卡 1:1** — 不同卡片類型的換算比例不同。
> 5. **有代價的能力 = 0V** — 代價抵消價值，不額外計價。
> 6. **武器彈藥打完進棄牌堆** — 不再使用短休息/長休息回復。
