## 使用情境

### A. 上傳給 claude.ai
1. 把整個資料夾打包成 zip
2. 上傳到 claude.ai 對話
3. 開頭丟「請先讀 `規則書索引_v07_26042606.md` 了解結構」
4. 後續可問任何規則問題

- 直接用文字編輯器或 markdown viewer 開
- `規則書索引_v07_26042606.md` 是入口,從那邊找到要讀的章節
