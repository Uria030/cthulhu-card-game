## 一、成長體系總覽｜Growth Overview

### 1.1 五條成長路徑

| 路徑 | 貨幣 | 產出 | 解鎖條件 |
|------|------|------|---------|
| 卡片升級 | 經驗值 | 升級版卡片（LV0→LV5） | 無，基礎功能 |
| 天賦樹 | 天賦點 | 被動能力節點 | 無，基礎功能 |
| 鍛造 | 素材 + 資源 | 資產卡附加強化詞條 | 需團隊精神「鍛造解鎖」 |
| 製作 | 素材 + 資源 | 一張新的完整卡片 | 需團隊精神「製作解鎖」 |
| 團隊精神 | 凝聚力 | 隊伍共用能力與系統解鎖 | 無，基礎功能 |

### 1.2 成長節奏

- **前期：** 資源有限，只能解鎖最急需的功能（例如鍛造或製作）
- **中期：** 開始展開多條路徑，調查員和隊伍風格逐漸成形
- **後期：** 點滿特定團隊精神解鎖里程碑，隊伍打法產生質變

### 1.3 成長必須在世界中贏得

所有成長資源來自場景探索（採集素材）、擊殺怪物（掉落素材）、完成章節（經驗值、天賦點、凝聚力），不能透過抽象選單獲取。

## 二、鍛造系統｜Forging System

### 2.1 定義

鍛造是用素材和資源**強化牌組內現有的資產卡**，為其附加一個強化詞條。

### 2.2 鍛造規則

| 項目 | 規格 |
|------|------|
| 可鍛造的卡片類型 | 僅資產卡（`asset`） |
| 不可鍛造 | 盟友（`ally`）、事件（`event`）、技能（`skill`）、臨時卡片（`is_temporary: true`） |
| 每張卡片最大鍛造次數 | 5 次（每個等級最多鍛造 1 次） |
| 鍛造次數上限突破 | 特定陣營能力可超過 5 次上限 |
| 鍛造效果持久性 | 永久附著，不因等級升級消失 |
| 鍛造觸發場景 | 長休息時（預設）、關卡中透過「鍛造卡」觸發 |
| 解鎖條件 | 需投資團隊精神「鍛造解鎖」 |

### 2.3 卡片等級的兩條升級路線

| 路線 | 費用 | 效果 | 性質 |
|------|------|------|------|
| 經驗值路線 | 經驗值 | 等級提升（LV0→LV5），獲得標準強化 | 正規升級 |
| 素材鍛造路線 | 素材 + 資源 | 在當前等級附加一個強化詞條 | 自由強化 |

兩條路線可以疊加。鍛造效果永久附著，不因等級升級消失。

### 2.4 鍛造詞條價值計算原則

> **鍛造詞條的價值為面值，不乘以使用次數。**

這是對玩家的優待。一把 3 發使用次數的手槍鍛造「傷害 +1」，價值算 1V 而非 3V。

## 三、素材系統｜Material System

### 3.1 五種素材類別

| 類別 | 代碼 | 來源 | 主題 |
|------|------|------|------|
| 礦物 | `mineral` | 場景採集 | 堅硬、鋒利、防護 |
| 木材 | `wood` | 場景採集 | 結構、支撐、效率 |
| 蟲類 | `insect` | 場景採集 | 毒素、寄生、腐蝕 |
| 魚類 | `fish` | 場景採集 | 滑溜、適應、恢復 |
| 怪物素材 | `monster` | 擊殺怪物掉落 | 超自然、力量、恐懼 |

### 3.2 素材等級與價值

每種類別有 10 個等級（LV1 至 LV10）。

| 素材等級 | 素材價值（SV） |
|----------|---------------|
| LV1–2 | 1 SV |
| LV3–4 | 2 SV |
| LV5–6 | 3 SV |
| LV7–8 | 5 SV |
| LV9–10 | 8 SV |

### 3.3 鍛造費用計算

> **所需素材數量 = 詞條價值（V）÷ 素材價值（SV）**（向上進位）

同一次鍛造只能使用同一類別的素材。鍛造還需額外花費資源作為固定手續費（具體數值待定）。

### 3.4 怪物素材掉落規則

| 敵人位階 | 素材等級範圍 | 掉落數量 |
|----------|-------------|----------|
| 雜兵 | LV1–3 | 1 個 |
| 威脅 | LV3–5 | 1–2 個 |
| 精英 | LV5–7 | 2–3 個 |
| 頭目 | LV7–9 | 3–5 個 |
| 巨頭 | LV9–10 | 5+ 個 |

怪物素材類別由怪物家族決定（例如深潛者 → 魚類素材，米戈 → 蟲類素材）。

## 五、製作系統｜Crafting System

### 5.1 定義

製作是用素材和資源**創造一張新的完整卡片**。

### 5.2 製作規則

| 項目 | 規格 |
|------|------|
| 產出 | 一張完整的三合一卡片 |
| 產出等級 | 固定 LV0 |
| 產出稀有度 | 由配方決定 |
| 費用 | 指定素材 + 資源 |
| 觸發場景 | 長休息時（預設）、關卡中透過「製作卡」觸發 |
| 解鎖條件 | 需投資團隊精神「製作解鎖」 |

### 5.3 配方系統

| 項目 | 規格 |
|------|------|
| 配方來源 | 預設隱藏，透過場景探索或陣營升級解鎖 |
| 配方內容 | 指定產出卡片、所需素材類別與等級、所需資源數量 |

### 5.4 產出卡片的兩種性質

| 性質 | 代碼 | 可鍛造 | 素材需求 | 配方解鎖難度 |
|------|------|--------|----------|-------------|
| 既有卡片 | `is_temporary: false` | 可以 | 較高 | 較難 |
| 臨時卡片 | `is_temporary: true` | 不可以 | 較低 | 較易 |

## 六、凝聚力系統｜Cohesion System

### 6.1 基礎規則

| 項目 | 規格 |
|------|------|
| 資源性質 | 隊伍共用資源，非個人資源 |
| 獲取量 | 固定，不按玩家人數縮放 |
| 獲取來源 | 長休息（每次 +1）、支線通關 |
| 用途 | 投資團隊精神 |

### 6.2 隊長制

| 項目 | 規格 |
|------|------|
| 單人遊戲 | 玩家自己是隊長 |
| 多人遊戲 | 創立遊戲大廳的玩家是隊長 |
| 團隊紀錄 | 存在隊長本地端，可存多筆 |
| 紀錄匹配 | 系統根據當前玩家組合自動匹配對應紀錄 |
| 舊紀錄 | 保留，原班人馬回來可讀回 |

### 6.3 玩家變動規則

| 情境 | 處理方式 |
|------|---------|
| 玩家換調查員 | 不影響團隊紀錄 |
| 隊伍成員變動未超過一半 | 繼續使用當前紀錄 |
| 隊伍成員變動超過一半 | 開啟新的團隊紀錄 |

> **核心判定：** 凝聚力綁定**玩家組合**而非角色組合。是現實中一起打的那群人在累積默契。

## 八、書籍子系統｜Book Subsystem

### 8.1 閱讀機制

| 項目 | 規格 |
|------|------|
| 閱讀方式 A | 純累積次數 — 花費行動點，每次累積 1 次進度 |
| 閱讀方式 B | 累積次數 + 智力檢定 — 成功才累積進度 |
| 所需次數 | 由卡片定義，通常較高 |
| 進度保留 | **跨章節保留** |
| 完成標記 | 達成所需次數後，記錄「已了解此書的知識」 |

### 8.2 書籍分級

| 分級 | 說明 | 解鎖需求 |
|------|------|---------|
| 普通書籍 | 日記、筆記、報紙剪報 | 無，直接可閱讀 |
| 神話典籍 | 《死靈之書》《妖蛆之秘密》等 | 需團隊精神「古文解讀」達到對應深度 |

### 8.3 古文解讀深度對應

| 深度 | 可解讀的書籍難度 | 代表書籍範例 |
|------|----------------|------------|
| 1 點 | 初階神話文獻 | 入門級翻譯文本、邪教筆記 |

## 九、遺跡子系統｜Relic Subsystem

### 9.1 定位

遺跡與書籍是平行的兩套知識獲取管道。

| 面向 | 書籍 | 遺跡 |
|------|------|------|
| 載體 | 文字典籍 | 石板、古物、雕像、首飾 |
| 知識路線 | 閱讀、翻譯、理解 | 觸摸、拼接、感應 |
| 解鎖管道 | 知識與神話類團隊精神（古文解讀） | 調查與資訊類團隊精神（遺跡解讀） |
| 主題調性 | 學者在燈下翻書 | 考古學家在遺址挖掘 |

### 9.2 研究機制

與書籍閱讀機制完全平行：純累積次數或累積次數 + 智力檢定，進度跨章節保留，完成後長休息時購買升級版（內容事先不公開）。

### 9.3 遺跡分級

| 分級 | 說明 | 解鎖需求 |
|------|------|---------|
| 普通遺物 | 一般古物、裝飾品 | 無，直接可研究 |
| 神話遺跡 | 帶有超自然力量的古代遺物 | 需團隊精神「遺跡解讀」達到對應深度 |

## 十、鍛造計算範例｜Forging Examples

### 範例一：用 LV5 礦物鍛造「利刃 +1」

| 項目 | 數值 |
|------|------|
| 詞條 | 利刃 +1（傷害 +1） |
| 詞條價值 | 1V |
| 素材等級 | LV5（SV = 3） |
| 所需素材數量 | 1V ÷ 3SV = 0.33 → **1 顆** |

### 範例二：一把手槍的完整鍛造歷程

| 等級 | 經驗值升級效果 | 鍛造詞條（玩家選擇） |
|------|-------------|-------------------|
| LV0 | 基礎：傷害 2、力量 +1 | 利刃 +1（礦物）→ 傷害變 3 |
| LV1 | 傷害 +1 | 蟲淬 II — 中毒（蟲類）→ 命中施加中毒 |
| LV2 | 力量 +1 | 補給 +1（木材）→ 使用次數 +1 |
| LV3 | 雙屬性修正 | 銳化 +1 敏捷（怪物素材） |
| LV4 | all +1 或 力量 +3 | 元素附魔 — 火（怪物素材） |
| LV5 | 最終形態 | 已達鍛造上限 5 次 |

每個玩家的同一把手槍因為鍛造選擇不同而長得完全不一樣。
