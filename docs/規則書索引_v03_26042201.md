## §4 覆蓋規則（V 值公式更新）

**適用檔案**：`03_rulebook_ch3.md`（第三章 · 卡片系統總綱）

規則書 v1.0 第三章本文中的 V 值計算公式為**待更新文字**，**以下列新版為準**：

- **所有卡片類型統一為 1:1**（含資產卡；原 2:1 作廢）
- **事件卡、盟友卡、技能卡**亦為 1:1；技能卡費用固定為 0
- **等級抵扣新階梯**：LV1 -0.5V / LV2 -1V / LV3 -2V / LV4 -3V / LV5 -4V（**非**舊版 -1V/-2V/-3V/-4V/-5V）
- 驗證器、Prompt、程式碼以 1:1 為準
- 若看到第三章文字中的 2:1 公式，**視為待更新文字，不要採用**

**權威依據**：
- `s04_card_value_spec.md` 卡片價值計算規範 v1.1 修正案
- `docs/04_SystemPrompt模板庫_卡片設計篇.md`
- 記憶 `project_v_value_formula.md`

## §5 給 claude.ai 使用者的指示

1. **上傳這份索引** → claude.ai 理解規則書結構
2. **依需求額外上傳對應 `.md`**：
   - 卡片數值/機制 → `03_rulebook_ch3.md` + `s04_card_value_spec.md`
   - **陣營體系 / 構築** → **`s05_faction_pillars.md`（新增）**
   - 怪物/敵人 → `s01_monster_family.md` + `06_rulebook_ch6.md`
   - 場景/關卡 → `05_rulebook_ch5.md`
   - 譯名/命名 → `s02_mythos_terms.md`
3. 告知 §4 覆蓋規則，否則會讀到舊 2:1 公式
