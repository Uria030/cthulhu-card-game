## 一、設計哲學

### 雙軌檢定的靈魂

混沌袋是支柱 2（雙軌檢定）的核心機制。骰子給予掌控感，混沌袋剝奪掌控感。

| | 物理攻擊（d20 軌） | 法術攻擊（混沌袋軌） |
|--|-------------------|-------------------|
| 類比 | D&D 一般攻擊：玩家擲命中骰 | D&D 高階法術：法術一定發動，對方做豁免 |
| 不確定性 | 「能不能打到？」 | 「代價多大？」 |
| 命中 | 需要擲骰達到 DC | **一定命中，傷害一定造成** |
| 代價 | 消耗彈藥等資源 | 抽混沌袋承受副作用 |
| 成長感 | 修正值越來越高，命中率上升 | 副作用管理越來越熟練，但命運不可控 |
| 情感 | 「我在進步」的可靠感 | 「我在玩火」的不安感 |

### 法術永遠命中

法術釋放出去**一定造成效果**。混沌袋不決定「有沒有命中」，而是決定「施法者承受什麼副作用」。這是克蘇魯世界觀的核心 — 禁忌力量確實強大，但使用它的代價永遠不可預測。

### 怪物法術防禦值

每隻怪物有一個**法術防禦值（Spell Defense, 0–9）**。

- 抽出的混沌袋標記數值 ≥ 法術防禦值 → 副作用**減輕或取消**
- 抽出的混沌袋標記數值 < 法術防禦值 → 副作用**全額發動**
- 法術傷害本身**不受法術防禦值影響** — 傷害永遠造成

設計意圖：法術防禦值高的怪物（如舊日支配者的眷屬）不會讓你的法術無效，但會讓你施法的代價更加慘烈。你必須決定：值不值得對這種怪物用法術？

## 二、混沌袋標記分類

混沌袋中的標記分為五大類：

| 分類 | 標記 | 說明 |
|------|------|------|
| 數字標記 | +1、0、-1、-2、-3、-4 | 純數值修正 |
| 情境標記 | 骷髏、邪教徒、石版、遠古邪物 | 由關卡定義數值與場景效果（負面為主） |
| 神話標記 | 怪物、毀滅、次元門、線索、頭條 | 由關卡定義數值與場景效果（涵蓋正面到負面的完整光譜） |
| 極端標記 | 觸手（自動最壞）、遠古印記（好運） | 固定規則 |
| 動態標記 | 祝福、詛咒 | 遊戲中被卡片效果加入或移除 |

## 三、數字標記

### 3.1 標記列表

| 標記 | 數值 | 說明 |
|------|------|------|
| `+1` | +1 | 輕微好運 |
| `0` | 0 | 無修正 |
| `-1` | -1 | 輕微厄運 |
| `-2` | -2 | 中等厄運 |
| `-3` | -3 | 嚴重厄運 |
| `-4` | -4 | 極度厄運（僅高難度或後期關卡） |

### 3.2 數字標記的作用

數字標記**只有數值**，沒有場景效果。

抽到數字標記後，將數值與怪物法術防禦值比較：
- 數值 ≥ 法術防禦值 → 施法順利，無副作用
- 數值 < 法術防禦值 → 差值決定副作用程度（具體規則見第六章）

### 3.3 難度配置

關卡設計師透過調整數字標記的數量和分佈來控制難度：

| 難度 | 典型配置 |
|------|---------|
| 簡單 | +1 ×2、0 ×3、-1 ×2 |
| 標準 | +1 ×1、0 ×2、-1 ×2、-2 ×2 |
| 困難 | +1 ×1、0 ×1、-1 ×2、-2 ×2、-3 ×1 |
| 專家 | 0 ×1、-1 ×2、-2 ×2、-3 ×2、-4 ×1 |

> 以上為參考配置，關卡設計師可完全自訂。

## 四、情境標記（核心設計 — 雙重效果）

情境標記是混沌袋的精華。每個情境標記**同時產生兩個效果**：

1. **數值效果** — 一個負值修正，由關卡設計師定義（範圍 -1 至 -6）
2. **場景效果** — 一個改變場上狀態的事件，由關卡設計師從預設效果中選擇

**主題：** 死亡、傷害、肉體侵蝕

| 項目 | 定義方式 |
|------|---------|
| 數值 | 由關卡設計師定義（建議 -1 至 -3） |
| 場景效果 | 從以下選擇一項（關卡設計師配置）： |

**可選場景效果：**

| 效果代碼 | 中文 | 說明 |
|---------|------|------|
| `skull_damage` | 死亡之觸 | 施法者受到 X 點 HP 傷害 |
| `skull_trauma` | 創傷侵蝕 | 施法者 HP 上限 -1（極端，僅限高難度） |
| `skull_bleed` | 血祭 | 施法者獲得流血 X 狀態 |
| `skull_weaken` | 生命流逝 | 施法者獲得無力狀態直到回合結束 |
| `skull_consume` | 生命代價 | 施法者失去場上一張盟友或資產（由施法者選擇） |

**設計意圖：** 骷髏代表法術對施法者肉體的反噬。你用禁忌力量，力量也在吞噬你的身體。

## 五、神話標記（來自 Arkham Horror 3rd Edition）

神話標記是混沌袋中涵蓋**正面到負面完整光譜**的標記。與情境標記不同，神話標記的場景效果是固定的（不由關卡設計師選擇），但數值仍由關卡設計師定義。

抽到神話標記不再只是「壞事發生」— 線索和頭條可能帶來意外收穫，讓施法充滿戲劇性的不確定感。

---

### 5.1 線索（Clue）⭐ 正面標記

**代碼：** `clue`

| 項目 | 規則 |
|------|------|
| 數值 | 由關卡設計師定義（建議 **-1 至 0**） |
| 場景效果 | 在施法者所在板塊**生成 1 個線索** |

**設計意圖：** 法術撕開了現實的帷幕，你無意間瞥見了隱藏的真相。這是施法的意外收穫 — 代價很小，還額外獲得了調查進度。對調查與預言類法術特別有主題共鳴。

---

### 5.2 頭條（Headline）⭐ 中性標記

**代碼：** `headline`

| 項目 | 規則 |
|------|------|
| 數值 | 由關卡設計師定義（建議 **-1 至 -2**） |
| 場景效果 | **抽取 1 張遭遇卡**，立即結算 |

**設計意圖：** 你的法術引起了漣漪，但漣漪的方向不可預測。遭遇卡可能是壞事（怪物出現、陷阱觸發），也可能是機會（NPC 現身、劇情推進）。頭條代表「你的行為引起了世界的注意，但世界的回應是不確定的」。

---

### 5.3 怪物（Monster）💀 負面標記

**代碼：** `monster`

| 項目 | 規則 |
|------|------|
| 數值 | 由關卡設計師定義（建議 **-2 至 -3**） |
| 場景效果 | 在施法者所在板塊或相鄰板塊**生成 1 隻敵人**（生成的怪物種類由關卡定義） |

**設計意圖：** 你的法術是一盞燈，而黑暗中的東西被光吸引過來了。施法行為本身就是對異界的呼喊，而某些東西聽到了。生成的怪物種類應與關卡的怪物家族一致。

**設計意圖：** 你使用禁忌力量，加速了舊日支配者降臨的進程。每一次施法都在推動世界走向終末。毀滅標記是最直接的時間壓力 — 你越依賴法術，末日就越近。

## 六、極端標記

### 6.1 觸手（Tentacle / Auto-Worst）

**代碼：** `tentacle`

| 項目 | 規則 |
|------|------|
| 數值 | **自動視為最低** — 無論法術防禦值是多少，副作用全額發動 |
| 場景效果 | **同時觸發所有情境標記中最嚴重的場景效果** |
| 額外效果 | 施法者受到 1 點 SAN 傷害（觸碰了不該觸碰的深淵） |

**設計意圖：** 觸手是混沌袋的終極恐怖。當你抽到它，代表禁忌力量完全失控，一切最壞的事同時發生。這是克蘇魯世界觀中「人類妄圖操控不可名狀之物」的終極懲罰。

> **袋中數量：** 固定 1 個，任何難度都有，不可移除。

---

### 6.2 遠古印記（Elder Sign）

**代碼：** `elder_sign`

| 項目 | 規則 |
|------|------|
| 數值 | **自動視為最高** — 副作用完全取消 |
| 場景效果 | **無副作用**，且觸發施法者的調查員卡上定義的個人效果 |
| 額外效果 | 施法者恢復 1 點 SAN（與宇宙秩序短暫共鳴） |

**設計意圖：** 遠古印記是黑暗中的一絲微光。在這一瞬間，法術完美地按照施法者的意志運作，沒有任何代價。但這種好運極其罕見 — 它提醒你為什麼繼續冒險：因為偶爾，命運會站在你這邊。

> **袋中數量：** 固定 1 個，任何難度都有，不可移除。

## 七、動態標記

### 7.1 祝福（Bless）

**代碼：** `bless`

| 項目 | 規則 |
|------|------|
| 數值 | **+1** |
| 額外效果 | **再抽一顆**（連鎖抽取） |
| 抽後處理 | **移出混沌袋**，放回標記池（不放回袋中） |
| 袋中上限 | **10 個** |

### 7.2 詛咒（Curse）

**代碼：** `curse`

| 項目 | 規則 |
|------|------|
| 數值 | **-1** |
| 額外效果 | **再抽一顆**（連鎖抽取） |
| 抽後處理 | **移出混沌袋**，放回標記池（不放回袋中） |
| 袋中上限 | **10 個** |

### 7.3 祝福與詛咒的互動

- 同一次抽取中同時抽到祝福和詛咒 → **互相抵消，兩者都移除**。
- 祝福和詛咒由卡片效果加入混沌袋，不在關卡初始配置中。
- 強化（`empowered`）與弱化（`weakened`）狀態**不影響**混沌袋抽取。

## 八、法術攻擊完整結算流程

```
施法者宣告施放法術
  ↓
確認法術卡的費用已支付（資源、橫置、使用次數等）
  ↓
法術效果一定生效 — 造成卡面定義的傷害/效果
  ↓
抽混沌袋 1 顆標記
  ↓
判定標記類型：

┌─ 遠古印記 → 無副作用 + 觸發調查員個人效果 + 恢復 1 SAN
│
├─ 祝福 → +1 + 再抽一顆（連鎖）+ 移出袋
│
├─ 數字標記 → 取得數值
│
├─ 情境標記 → 取得數值 + 記錄場景效果（負面）
│
├─ 神話標記 → 取得數值 + 記錄場景效果（正面/中性/負面）
│
├─ 詛咒 → -1 + 再抽一顆（連鎖）+ 移出袋
│
└─ 觸手 → 自動最壞結果（見 6.1）

  ↓
將最終數值與目標怪物的法術防禦值比較：

  數值 ≥ 法術防禦值 → 副作用減輕或取消
  數值 < 法術防禦值 → 副作用全額發動
    └ 差值越大，副作用越嚴重（可選規則，由場景定義）

  ↓
結算場景效果（若有情境標記或神話標記）
  注意：神話標記的場景效果無論副作用判定結果如何都一定發動
        （線索生成、遭遇卡抽取、怪物生成、毀滅推進、次元門開啟）
  ↓
法術結算完成
```

## 九、副作用減輕規則

### 9.1 基礎判定

| 比較結果 | 副作用處理 |
|---------|----------|
| 數值 ≥ 法術防禦值 | 場景效果**取消**，數值效果**取消** |
| 數值 < 法術防禦值，差值 1–2 | 場景效果**取消**，施法者受到 1 點 SAN 傷害（輕微反噬） |
| 數值 < 法術防禦值，差值 3–4 | 場景效果**全額發動** |
| 數值 < 法術防禦值，差值 5+ | 場景效果**全額發動** + 施法者額外受到 1 點 SAN 傷害 |

### 9.2 觸手的特殊處理

觸手標記**無視法術防禦值**，一律按最壞結果處理。

### 9.3 遠古印記的特殊處理

遠古印記**無視法術防禦值**，一律取消所有副作用。

### 9.4 神話標記的特殊處理

神話標記的場景效果**無視法術防禦值判定**，一律發動。法術防禦值只影響神話標記的數值部分（是否計入副作用傷害），但場景事件本身（生成線索、生成怪物、推進毀滅等）永遠觸發。

設計意圖：神話標記代表施法在世界上造成的漣漪，這些漣漪不是怪物能阻擋的 — 它們是宇宙層級的因果連鎖。

## 十、怪物法術防禦值設計指引

| 位階 | 建議法術防禦值 | 設計意圖 |
|------|-------------|---------|
| 雜兵 | 0–1 | 法術對雜兵幾乎沒有風險 |
| 威脅 | 1–3 | 輕微風險，大部分時候安全 |
| 精英 | 3–5 | 中等風險，需要考慮是否值得用法術 |
| 頭目 | 5–7 | 高風險，法術副作用頻繁發動 |
| 巨頭 | 7–9 | 極高風險，幾乎每次施法都要承受代價 |

**設計哲學：** 對低階怪物用法術像用大砲打蚊子 — 效果好但浪費。對高階怪物用法術是真正的賭博 — 傷害一定造成，但你自己也可能陪進去。

## 十一、混沌袋與關卡的關係

### 11.1 每關獨立配置

每個關卡（場景）有自己的**混沌袋配置表**，定義：

1. 數字標記的數量和面值
2. 每個情境標記的數量、數值、場景效果
3. 每個神話標記的數量和數值
4. 觸手和遠古印記固定各 1 個
5. 祝福和詛咒初始為 0（由卡片效果動態加入）

### 11.2 難度影響

同一關卡提供四種難度的混沌袋配置：

| 難度 | 數字標記傾向 | 情境標記數值 | 神話標記配置 | 情境標記場景效果 |
|------|------------|------------|------------|---------------|
| 簡單 | 偏正面 | -1 至 -2 | 線索 ×2、頭條 ×1、怪物 ×1 | 較輕微 |
| 標準 | 平衡 | -1 至 -3 | 線索 ×1、頭條 ×1、怪物 ×1、毀滅 ×1 | 標準 |
| 困難 | 偏負面 | -2 至 -4 | 線索 ×1、怪物 ×1、毀滅 ×1、次元門 ×1 | 較嚴重 |
| 專家 | 極負面 | -3 至 -6 | 怪物 ×1、毀滅 ×1、次元門 ×1 | 最嚴重 |

> 注意難度越高，正面神話標記（線索）越少，負面神話標記（次元門）越多。

### 11.3 怪物家族影響

不同怪物家族主導的關卡，混沌袋的情境標記場景效果應呼應家族主題：

| 家族 | 骷髏偏好 | 邪教徒偏好 | 石版偏好 | 遠古邪物偏好 |
|------|---------|----------|---------|------------|
| 克蘇魯 | 血祭（流血） | 信徒回應（深潛者增援） | 禁忌知識（SAN 傷害） | 異界滲透（鬧鬼） |
| 哈斯塔 | 生命流逝（無力） | 末日推進（Agenda） | 瘋狂低語（發瘋） | 時空扭曲（隨機傳送） |
| 莎布 | 死亡之觸（HP 傷害） | 儀式共鳴（怪物回血） | 精神枯竭（疲勞） | 裂隙擴張（次元門） |
| 奈亞 | 生命代價（失去盟友） | 暴露（失去隱蔽） | 不應知曉之事（神啟卡） | 空間斷裂（斷開連接） |
| 猶格 | 血祭（流血） | 末日推進（Agenda） | 記憶崩解（棄手牌） | 時空扭曲（隨機傳送） |
| 克圖格亞 | 死亡之觸（HP 傷害） | 暴露（失去隱蔽） | 禁忌知識（SAN 傷害） | 異界之火（失火） |
| 伊格 | 血祭（流血） | 信徒回應（蛇人增援） | 瘋狂低語（發瘋） | 異界滲透（鬧鬼） |

> 神話標記的場景效果是固定的（線索生成線索、怪物生成怪物等），不受家族影響。但怪物標記生成的具體怪物種類應與關卡的家族一致。

### 11.4 戰役中的混沌袋演變

混沌袋在戰役推進過程中會逐漸變差：

- **每一關結束後**，依據結果（A/B/C）可能在混沌袋中**永久新增負面標記**。
- 例如：第三關結果 C（最差結果）→ 混沌袋永久加入 1 個 -2 標記。
- 也可能新增神話標記：結果 C → 混沌袋永久加入 1 個毀滅標記。
- 這呼應支柱 4（三層死亡）的溫水煮青蛙設計 — 調查員的處境隨戰役推進逐漸惡化。

## 十三、六大法術類型框架

> 以下定義法術的六大機制分類，每張法術卡固定歸屬一個類型和一個施法專精。

### 12.1 戰鬥與毀滅類（Combat & Destruction）

| 項目 | 定義 |
|------|------|
| 核心功能 | 造成傷害（HP 和/或 SAN） |
| 傷害元素 | 神秘（Arcane）— 無抗性 |
| 副作用傾向 | 偏向骷髏（肉體反噬） — 用身體換取破壞力 |
| 與施法專精的關係 | 咒語（即時爆發）、引導（持續傷害）、煉金（範圍傷害） |
| 代表效果 | `deal_damage`、`deal_horror`、`add_status`（燃燒等） |

### 12.2 調查與預言類（Investigation & Prophecy）

| 項目 | 定義 |
|------|------|
| 核心功能 | 獲取資訊、發現線索、揭露隱藏 |
| 副作用傾向 | 偏向石版（精神侵蝕） — 知曉不該知道的事 |
| 與施法專精的關係 | 儀式（長時間占卜）、冥想（靈視）、引導（持續感應） |
| 代表效果 | `discover_clue`、`reveal_tile`、`modify_test`（感知加成） |

### 12.3 防護與迴避類（Protection & Evasion）

| 項目 | 定義 |
|------|------|
| 核心功能 | 保護自己或隊友、迴避危險 |
| 副作用傾向 | 偏向邪教徒（引起注意） — 防護結界像信號彈 |
| 與施法專精的關係 | 儀式（持久護盾）、咒語（即時閃避）、冥想（精神防壁） |
| 代表效果 | `add_status`（護甲、護盾）、`disengage_enemy`、`jump` |

### 12.4 時空與位面操控類（Spacetime & Planar）

| 項目 | 定義 |
|------|------|
| 核心功能 | 操控空間、傳送、封印、開啟/關閉門戶 |
| 副作用傾向 | 偏向遠古邪物（環境異變） — 扭曲時空的連鎖反應 |
| 與施法專精的關係 | 儀式（封印儀式）、引導（維持空間穩定）、咒語（瞬間傳送） |
| 代表效果 | `seal_gate`、`move_investigator`（傳送）、`connect_tiles`、`disconnect_tiles` |

### 12.5 召喚與束縛類（Summoning & Binding）

| 項目 | 定義 |
|------|------|
| 核心功能 | 召喚援軍、束縛敵人、控制怪物 |
| 副作用傾向 | 偏向邪教徒（黑暗勢力回應） — 召喚行為本身是雙面刃 |
| 與施法專精的關係 | 儀式（長時間召喚）、引導（維持束縛）、煉金（消耗品式召喚物） |
| 代表效果 | `spawn_enemy`（召喚盟友怪物）、`stun_enemy`、`engage_enemy`（強制束縛） |

### 12.6 治癒與淨化類（Healing & Purification）

| 項目 | 定義 |
|------|------|
| 核心功能 | 恢復 HP/SAN、移除負面狀態、淨化詛咒 |
| 副作用傾向 | 偏向骷髏（以己之血換人之命） — 治療者承受代價 |
| 與施法專精的關係 | 冥想（精神治療）、儀式（大規模淨化）、煉金（治療藥劑） |
| 代表效果 | `heal_hp`、`heal_san`、`remove_status`、`remove_curse` |

| 項目 | 決議 |
|------|------|

> **給未來 Claude 實例的備註｜Note to Future Claude Instances**
>
> 本文件的所有規則均經過與專案原創者 Uria 的逐項討論確認。
> 特別注意以下設計精神：
>
> 1. **法術一定命中** — 這是最核心的設計。混沌袋不決定命中，決定代價。不要寫出「法術未命中」的邏輯。
> 2. **神秘元素無抗性** — 任何怪物都不能對神秘元素有抗性或免疫。法術防禦值影響的是副作用，不是傷害。
> 3. **情境標記有雙重效果** — 一個數值 + 一個場景效果，缺一不可。不要把情境標記設計成只有數值。
> 4. **神話標記的場景效果一定發動** — 無論法術防禦值判定結果如何，神話標記的場景事件（生成線索、生成怪物等）永遠觸發。法術防禦值只影響數值部分。
> 5. **神話標記有正面到負面的完整光譜** — 線索是好事、頭條不確定、怪物/毀滅/次元門是壞事。施法不再只有「無事」和「壞事」兩種結果。
> 6. **觸手和遠古印記各 1 個** — 固定數量，任何難度都有，不可移除。
> 7. **法術防禦值不公開** — 符合 E-4 隱藏資訊設計。玩家透過嘗試和觀察推測。
> 8. **抽多顆是成長能力** — 基礎規則固定抽 1 顆。抽 2 顆選 1 是解鎖的強大能力。
> 9. **六大法術類型各有副作用傾向** — 這不是硬性規定，但關卡設計師應盡量遵循，讓玩家能預期風險方向。
> 10. **混沌袋隨戰役惡化** — 跟創傷累積一樣，是溫水煮青蛙的設計。戰役後期可能新增更多負面神話標記。
