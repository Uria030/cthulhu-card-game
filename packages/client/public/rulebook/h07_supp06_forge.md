## 一、設計哲學｜Design Philosophy

### 1.1 兩個系統的定位

| 系統 | 定義 | 核心動詞 |
|------|------|----------|
| **鍛造（Forging）** | 用素材 + 資源強化牌組內現有的資產卡 | 升級 |
| **製作（Crafting）** | 用素材 + 資源創造一張新卡片 | 創造 |

### 1.2 成長必須在世界中贏得

素材來自場景探索（採集自然資源）和擊殺怪物（怪物掉落），不能透過抽象選單獲取。玩家必須親自踏入危險的地方，才能取得強化自己的資源。

### 1.3 卡片等級的兩條升級路線

每張資產卡從 LV0 到 LV5 有兩條並行的成長路徑：

| 路線 | 費用 | 效果 | 性質 |
|------|------|------|------|
| **經驗值路線** | 經驗值 | 等級提升（LV0 → LV1 → ... → LV5），獲得該等級的標準強化 | 正規升級 |
| **素材鍛造路線** | 素材 + 資源 | 在當前等級附加一個強化詞條，提升卡片 Value 值 | 自由強化 |

兩條路線可以疊加。鍛造效果永久附著在卡片上，不因等級升級而消失。

## 二、素材系統｜Material System

### 2.1 五種素材類別

| 類別 | 中文 | 來源 | 主題 |
|------|------|------|------|
| `mineral` | 礦物 | 場景採集 | 堅硬、鋒利、防護 |
| `wood` | 木材 | 場景採集 | 結構、支撐、效率 |
| `insect` | 蟲類 | 場景採集 | 毒素、寄生、腐蝕 |
| `fish` | 魚類 | 場景採集 | 滑溜、適應、恢復 |
| `monster` | 怪物素材 | 擊殺怪物掉落 | 超自然、力量、恐懼 |

### 2.2 素材等級

每種類別有 10 個等級（LV1 至 LV10），等級越高越稀有、越強力。

### 2.3 素材價值（SV）

| 素材等級 | 素材價值（SV） |
|----------|---------------|
| LV1-2 | 1 SV |
| LV3-4 | 2 SV |
| LV5-6 | 3 SV |
| LV7-8 | 5 SV |
| LV9-10 | 8 SV |

### 2.4 鍛造費用計算

> **所需素材數量 = 詞條價值（V）÷ 素材價值（SV）**（向上進位）

同一次鍛造只能使用同一類別的素材。鍛造還需額外花費資源作為固定手續費（具體數值待定）。

## 三、鍛造系統｜Forging System

### 3.1 鍛造規則

| 項目 | 規格 |
|------|------|
| 可鍛造的卡片類型 | 僅資產卡（`asset`） |
| 不可鍛造 | 盟友（`ally`）、事件（`event`）、技能（`skill`） |
| 每張卡片最大鍛造次數 | 5 次（每個等級最多鍛造 1 次） |
| 鍛造次數上限突破 | 特定陣營能力可超過 5 次上限 |
| 鍛造效果持久性 | 永久附著，不因等級升級消失 |
| 鍛造觸發場景 | 長休息時（預設）、關卡中透過「鍛造卡」觸發 |
| 鍛造卡取得方式 | 特定陣營在等級提升時獲得 |

### 3.2 資產子類型分類

鍛造時需區分資產的子類型，不同子類型可接受的詞條不同：

| 子類型 | 說明 | 代表卡片 |
|--------|------|----------|
| 近戰武器 | 不消耗彈藥/充能，需交戰 | 小刀、大劍、拳套 |
| 遠程物理武器 | 消耗彈藥 | 手槍、步槍、散彈槍、弓弩 |
| 遠程法術武器 | 消耗充能 | 攻擊型法術 |
| 道具 | 有使用次數 | 急救箱、手電筒 |
| 魔法道具 | 有使用次數或被動效果 | 護身符、咒文卷軸 |
| 有 HP 的資產 | 自帶血條的裝備 | 防護衣 |

一張卡片可能同時屬於多個子類型（例如防護衣既是道具又是有 HP 的資產）。

## 四、強化詞條價值表｜Enhancement Affix Value Table

### 4.1 鍛造詞條的價值計算原則

> **鍛造詞條的價值為面值，不乘以使用次數。**

這是對玩家的優待。一把 3 發子彈的手槍鍛造「傷害 +1」，價值算 1V 而非 3V。

### 4.2 礦物類詞條（傷害、護甲、物理強化）

| 詞條 | 效果 | 適用子類型 | +1 價值 | +2 價值 | +3 價值 |
|------|------|-----------|---------|---------|---------|
| 利刃 | 傷害 +X | 近戰、遠程物理 | 1V | 2V | 3V |
| 堅固 | 資產 HP +X | 有 HP 的資產 | 0.5V | 1V | 1.5V |
| 護甲鍛造 | 使用時獲得護甲 X 層 | 全資產 | 3V | 6V | 9V |
| 反擊 | 被攻擊時攻擊者受到 X 點固定傷害 | 近戰、遠程物理 | 1V | 2V | 3V |
| 多重攻擊 | 額外攻擊 X 次 | 近戰、遠程物理 | 1.5V | 3V | 4.5V |

### 4.3 木材類詞條（續航、資源、效率）

| 詞條 | 效果 | 適用子類型 | +1 價值 | +2 價值 | +3 價值 |
|------|------|-----------|---------|---------|---------|
| 補給 | 彈藥/使用次數/充能 +X | 遠程物理、遠程法術、道具、魔法道具 | 0.5V | 1V | 1.5V |
| 輕量化 | 費用 -X | 全資產 | 1V | 2V | 3V |
| 快速 | 打出時不用行動點 | 全資產 | 1V（固定） | — | — |
| 附加：抽牌 | 使用時抽 X 張牌 | 全資產 | 1V | 2V | 3V |
| 附加：回收 | 進棄牌堆時回收 X 張棄牌堆的卡 | 全資產 | 1.5V | 3V | 4.5V |

### 4.4 蟲類詞條（負面狀態附加）

> **所有狀態固定施加 1 層，不可增加層數。階級差異體現在可選狀態的強度。**

| 詞條階級 | 可選狀態 | 價值 | 適用子類型 |
|----------|---------|------|-----------|
| 蟲淬 I | 流血（2V）、脆弱（2V）、潮濕（1V） | 1V - 2V | 近戰、遠程物理、遠程法術 |
| 蟲淬 II | 中毒（3V）、燃燒（3V）、冷凍（3V）、弱化（3V） | 3V | 近戰、遠程物理、遠程法術 |
| 蟲淬 III | 繳械（4V）、疲勞（4V）、沈默（4V） | 4V | 近戰、遠程物理、遠程法術 |

鍛造時選擇具體狀態，之後固定在卡片上。

### 4.5 魚類詞條（回復、防禦、正面狀態）

| 詞條 | 效果 | 適用子類型 | +1 價值 | +2 價值 | +3 價值 |
|------|------|-----------|---------|---------|---------|
| 修補 | 使用時回復 X HP | 全資產 | 1.5V | 3V | 4.5V |
| 安撫 | 使用時回復 X SAN | 全資產 | 1.5V | 3V | 4.5V |
| 韌性 | 使用時取消 X 點傷害 | 全資產 | 0.5V | 1V | 1.5V |
| 附加：護盾 | 使用時獲得護盾 X 層 | 全資產 | 6V | 12V | 18V |
| 附加：再生 | 使用時獲得再生 X 層 | 全資產 | 6V | 12V | 18V |

### 4.6 怪物素材類詞條（檢定、元素、恐懼）

| 詞條 | 效果 | 適用子類型 | +1 價值 | +2 價值 | +3 價值 |
|------|------|-----------|---------|---------|---------|
| 銳化 | 單屬性檢定 +X | 近戰、遠程物理、遠程法術 | 0.5V | 1.5V | 3V |
| 恐嚇 | 命中時造成 X 點恐懼傷害 | 近戰、遠程物理、遠程法術 | 3V | 6V | 9V |
| 附加：元素附魔 | 攻擊額外附加指定元素（火/冰/電/神秘） | 近戰、遠程物理、遠程法術 | 神秘 3V，其他 2V（固定） | — | — |
| 附加：隱蔽 | 使用時獲得隱蔽 1 層 | 全資產 | 6V（固定） | — | — |
| 附加：強化 | 使用時獲得強化 X 層 | 全資產 | 3V | 6V | 9V |

元素附魔不替換原有元素，而是額外附加。一把物理手槍附魔火元素後，攻擊同時造成物理和火屬性傷害。

## 五、製作系統｜Crafting System

### 5.1 製作規則

| 項目 | 規格 |
|------|------|
| 產出 | 一張完整的三合一卡片（打出效果 + 屬性圖示 + 消費功能） |
| 產出卡片等級 | 固定 LV0 |
| 產出卡片稀有度 | 由配方決定 |
| 產出卡片類型 | 不限（資產、事件皆可，依配方設計） |
| 費用 | 指定素材 + 資源 |
| 觸發場景 | 長休息時（預設）、關卡中透過「製作卡」觸發 |
| 製作卡取得方式 | 特定陣營在等級提升時獲得 |

### 5.2 配方系統

| 項目 | 規格 |
|------|------|
| 配方來源 | 預設隱藏，透過場景探索解鎖或陣營升級解鎖 |
| 配方內容 | 指定產出卡片、所需素材類別與等級、所需資源數量 |
| 配方清單 | 在系統內預先設計，玩家從已解鎖的配方中選擇 |

### 5.3 產出卡片的兩種性質

| 性質 | 說明 | 可鍛造 | 素材需求 | 配方解鎖難度 |
|------|------|--------|----------|-------------|
| **既有卡片** | 卡片資料庫內正式設計的卡片 | 可以 | 較高 | 較難（探索、陣營升級） |
| **臨時卡片** | 製作專用的簡易卡片 | 不可以 | 較低 | 較易（基礎配方可能預設解鎖） |

資料結構上以 `is_temporary` 布林欄位區分。鍛造系統在執行時檢查此標記，拒絕對臨時卡片進行鍛造。

### 5.4 製作配方設計指引

每個配方在設計時需定義：

| 欄位 | 說明 |
|------|------|
| 配方代碼 | 唯一識別碼 |
| 產出卡片代碼 | 對應 card_definitions 的卡片 |
| 是否為臨時卡片 | `is_temporary` |
| 所需素材 | 類別、等級、數量的陣列 |
| 所需資源 | 資源數量 |
| 解鎖條件 | 預設解鎖 / 探索解鎖 / 陣營升級解鎖 |

配方範例：

```
配方：燃燒彈
產出：core_incendiary_round（事件卡，臨時）
素材：礦物 LV3 ×2 + 蟲類 LV2 ×1
資源：2
解鎖：S 陣營 LV2 升級
```

```
配方：.45 自動手槍
產出：core_45_automatic（資產卡，既有）
素材：礦物 LV5 ×3 + 木材 LV3 ×2
資源：5
解鎖：場景探索（武器工坊）
```

## 六、給編輯器的設計指引｜Editor Design Guidelines

### 6.1 怪物編輯器（MOD-03）

每隻怪物在設計時需標註：

| 欄位 | 說明 |
|------|------|
| 掉落素材類別 | 五類中的一種或多種 |
| 掉落素材等級 | 依敵人階層對應 |
| 掉落數量 | 依敵人階層對應 |

**敵人階層與掉落對應原則：**

| 敵人階層 | 素材等級範圍 | 掉落數量 |
|----------|-------------|----------|
| 雜兵（Minion） | LV1-3 | 1 個 |
| 威脅（Threat） | LV3-5 | 1-2 個 |
| 精英（Elite） | LV5-7 | 2-3 個 |
| 頭目（Boss） | LV7-9 | 3-5 個 |
| 巨頭（Titan） | LV9-10 | 5+ 個 |

**怪物素材類別由怪物家族決定：**
- 深潛者 → 魚類素材
- 米戈 → 蟲類素材
- 食屍鬼 → 怪物素材
- 具體對應在怪物家族設計時標註

### 6.2 場景編輯器（MOD-07）

每個採集點在設計時需標註：

| 欄位 | 說明 |
|------|------|
| 素材類別 | 礦物 / 木材 / 蟲類 / 魚類 |
| 素材等級 | 依場景難度設定 |
| 採集檢定 | 需要的屬性與 DC |
| 採集行動點 | 花費的行動點數 |

### 6.3 卡片設計器（MOD-01）

卡片設計時需新增以下欄位：

| 欄位 | 說明 |
|------|------|
| `is_temporary` | 是否為臨時卡片（製作專用） |
| `forgeable` | 是否可被鍛造（預設：資產卡 = true，其他 = false） |

## 七、鍛造計算範例｜Forging Calculation Examples

### 範例一：用 LV5 礦物鍛造「利刃 +1」

| 項目 | 數值 |
|------|------|
| 詞條 | 利刃 +1（傷害 +1） |
| 詞條價值 | 1V |
| 素材類別 | 礦物 |
| 素材等級 | LV5（SV = 3） |
| 所需素材數量 | 1V ÷ 3SV = 0.33 → 向上進位 = **1 顆** |
| 額外資源 | 手續費（待定） |

### 範例二：用 LV3 蟲類鍛造「蟲淬 II — 中毒」

| 項目 | 數值 |
|------|------|
| 詞條 | 蟲淬 II — 中毒（命中施加中毒 1 層） |
| 詞條價值 | 3V |
| 素材類別 | 蟲類 |
| 素材等級 | LV3（SV = 2） |
| 所需素材數量 | 3V ÷ 2SV = 1.5 → 向上進位 = **2 顆** |
| 額外資源 | 手續費（待定） |

### 範例三：用 LV9 怪物素材鍛造「附加：元素附魔 — 神秘」

| 項目 | 數值 |
|------|------|
| 詞條 | 元素附魔 — 神秘（攻擊額外附加神秘元素） |
| 詞條價值 | 3V |
| 素材類別 | 怪物素材 |
| 素材等級 | LV9（SV = 8） |
| 所需素材數量 | 3V ÷ 8SV = 0.375 → 向上進位 = **1 顆** |
| 額外資源 | 手續費（待定） |

### 範例四：一把手槍的完整鍛造歷程

**.45 自動手槍從 LV0 到 LV5 的成長路徑：**

| 等級 | 經驗值升級效果 | 鍛造詞條（玩家選擇） |
|------|-------------|-------------------|
| LV0 | 基礎：傷害 2、力量 +1 | 利刃 +1（礦物）→ 傷害變 3 |
| LV1 | 傷害 +1（標準版傷害 3） | 蟲淬 II — 中毒（蟲類）→ 命中施加中毒 |
| LV2 | 力量 +1（標準版力量 +2） | 補給 +1（木材）→ 彈藥 +1 |
| LV3 | 雙屬性修正（力量 +1、敏捷 +1） | 銳化 +1 敏捷（怪物素材）→ 敏捷額外 +1 |
| LV4 | all +1 或 力量 +3 | 元素附魔 — 火（怪物素材）→ 附加火屬性 |
| LV5 | 最終形態 | 已達鍛造上限 5 次 |

最終這把手槍：傷害 3+、力量 +3、敏捷 +2、命中施加中毒、彈藥 +1、附加火元素。
每個玩家的同一把手槍因為鍛造選擇不同而長得完全不一樣。

> 3. **每張卡片最多鍛造 5 次，每個等級 1 次** — 特定陣營能力可突破此上限。
> 4. **鍛造效果永久附著** — 不因等級升級消失。
> 5. **兩條升級路線疊加** — 經驗值路線（等級升級）+ 素材鍛造路線（詞條附加），同一張卡可以兩條都走。
> 6. **蟲類狀態固定 1 層** — 不能用鍛造給多層異常狀態。
> 7. **元素附魔是附加不是替換** — 物理武器附魔火元素後同時造成物理和火屬性傷害。
> 8. **素材類別決定強化方向** — 礦物偏物理強化、木材偏效率、蟲類偏控場、魚類偏回復、怪物素材偏檢定和超自然。
> 9. **製作分兩種產出** — 既有卡片（可鍛造）和臨時卡片（不可鍛造），用 `is_temporary` 區分。
> 10. **配方預設隱藏** — 玩家透過探索或陣營升級解鎖配方。
