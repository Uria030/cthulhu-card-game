## 全局設計原則

### 神秘元素定位

**神秘（Arcane）是遊戲中最強的攻擊元素。**

- 沒有任何怪物家族對神秘有抗性或免疫。
- 所有家族的元素抗性設計中，神秘一律留空。
- 這呼應支柱 2 雙軌檢定的設計意圖：神秘攻擊使用混沌袋（不可控），代價是結果不可預測且可能反噬。高風險換來的是無抗性的穿透力。
- 調查員要獲得神秘攻擊手段，必須走施法路線，承受 SAN 風險。

### 負面狀態速查

| 狀態 | 代碼 | 效果摘要 |
|------|------|---------|
| 中毒 | `poison` | 受到的傷害 +X |
| 流血 | `bleed` | 回合結束時扣 X 點 HP |
| 燃燒 | `burning` | 回合開始扣 HP，火攻增傷，移除潮濕 |
| 冷凍 | `frozen` | 移動花費 +1，冰攻增傷 |
| 黑暗 | `darkness` | 攻擊命中 -2 |
| 繳械 | `disarm` | 不能用資產攻擊 |
| 毀滅 | `doom_status` | 回合結束受傷 |
| 疲勞 | `fatigue` | 不能抽牌和獲得資源 |
| 發瘋 | `madness` | 受到的恐懼 +X |
| 標記 | `marked` | 受到的傷害和恐懼都 +X |
| 脆弱 | `vulnerable` | 受到的物理傷害 +X |
| 沈默 | `silence` | 無法施法 |
| 無力 | `weakness_status` | 近戰傷害降低 |
| 潮濕 | `wet` | 電擊傷害 +X |
| 弱化 | `weakened` | 下次擲骰取差的 |

### 正面狀態速查

| 狀態 | 代碼 | 效果摘要 |
|------|------|---------|
| 強化 | `empowered` | 下次擲骰取好的 |
| 護甲 | `armor` | 降低物理傷害 |
| 護盾 | `ward` | 降低恐懼傷害 |
| 隱蔽 | `stealth` | 不觸發藉機攻擊 |
| 加速 | `haste` | 額外行動點 |
| 再生 | `regeneration` | 回合開始恢復 HP |

## 家族一：克蘇魯眷族（House of Cthulhu）

**代碼：** `house_cthulhu`
**主神：** 克蘇魯 — 沉睡者、拉萊耶之主
**核心眷屬：** 深潛者、星之眷族、達貢與海德拉

### 主題

海洋、沉睡、夢境入侵、潮汐般不可阻擋的壓迫。

### 戰鬥特色

| 項目 | 傾向 |
|------|------|
| 攻擊類型 | 物理為主 — 觸手纏繞、利爪撕裂、肉體壓制 |
| 傷害偏重 | HP 傷害為主，搭配中等 SAN 傷害 |
| 戰鬥節奏 | 慢但沉重 — 每一擊傷害高，攻擊頻率低 |
| 典型詞綴 | 巨大（Massive）、壓垮（Crush） |
| AI 偏好 | `nearest` — 潮水般直撲而來 |

### 風格卡池傾向

| 屬性 | 頻率 | 場景 |
|------|------|------|
| 力量 | 高 | 掙脫觸手纏繞、承受正面衝撞 |
| 體質 | 高 | 承受海水侵蝕、抵抗窒息 |
| 敏捷 | 中 | 閃避猛撲攻擊 |
| 意志 | 低 | 僅高階單位的夢境入侵 |

### 元素與抗性

| 項目 | 內容 |
|------|------|
| 攻擊元素 | 物理 |
| 弱點 | 火、電 |
| 抗性 | 冰抗性（海底生物耐寒） |
| 免疫 | 無 |

### 施放的負面狀態

| 狀態 | 使用場景 | 頻率 |
|------|---------|------|
| 潮濕（`wet`） | 海水浸泡、觸手黏液 — 為電擊弱點建立連鎖 | 高 |
| 脆弱（`vulnerable`） | 觸手纏繞後暴露弱點 | 中 |
| 發瘋（`madness`） | 僅高階單位的夢境入侵 | 低 |

### 使用的正面狀態（怪物自身）

| 狀態 | 使用場景 | 頻率 |
|------|---------|------|
| 護甲（`armor`） | 堅硬的鱗片和甲殼 | 高 |
| 再生（`regeneration`） | 星之眷族與克蘇魯本體的肉體再生 | 中（精英以上） |

### 狀態免疫

| 狀態 | 原因 |
|------|------|
| 冷凍（`frozen`） | 深海生物適應極低溫 |
| 潮濕（`wet`） | 本身就是水生生物 |

### 位階範圍

| 怪物 | 位階 |
|------|------|
| 深潛者（雜兵型） | 雜兵 — 威脅 |
| 深潛者（精英型） | 精英 |
| 星之眷族 | 精英 — 頭目 |
| 達貢 / 海德拉 | 頭目 |
| 克蘇魯 | 巨頭（不可擊敗） |

### 恐懼特色

| 恐懼半徑 | 恐懼值 | 設計意圖 |
|---------|--------|---------|
| 1–2（近） | 2–4（高） | 遠遠看不怕，近距離壓迫感極強 |

### 敵對家族

哈斯塔眷族（水 vs 風）

## 家族二：哈斯塔眷族（House of Hastur）

**代碼：** `house_hastur`
**主神：** 哈斯塔 — 無以名狀者、黃衣之王
**核心眷屬：** 黃衣之王、拜亞基、伊塔庫亞、羅伊格爾與札爾

### 主題

風、天空、瘋狂、藝術腐化、黃色符號。
一首詩、一齣戲、一個黃色的印記，就足以讓理智崩塌。

### 戰鬥特色

| 項目 | 傾向 |
|------|------|
| 攻擊類型 | SAN 攻擊為主 — 精神侵蝕、幻覺、瘋狂耳語 |
| 傷害偏重 | SAN 傷害為主，物理傷害為輔 |
| 戰鬥節奏 | 不正面交鋒 — 從遠處持續消磨理智 |
| 典型詞綴 | 飛行（Flying）、詛咒（Curse） |
| AI 偏好 | `lowest_san` — 專挑最脆弱的心靈 |

### 風格卡池傾向

| 屬性 | 頻率 | 場景 |
|------|------|------|
| 意志 | 高 | 抵抗精神侵蝕 |
| 感知 | 高 | 辨別真實與虛幻 |
| 智力 | 中 | 理解不該理解的知識而不崩潰 |
| 力量 | 低 | 僅拜亞基近距離抓取 |

### 元素與抗性

| 項目 | 內容 |
|------|------|
| 攻擊元素 | 物理（風系物理衝擊） |
| 弱點 | 物理（精神系存在的實體化弱點） |
| 抗性 | 冰抗性（風行者耐寒） |
| 免疫 | 無 |

> **注意：** 哈斯塔眷族**沒有**神秘抗性。神秘是遊戲最強元素，無例外。

### 施放的負面狀態

| 狀態 | 使用場景 | 頻率 |
|------|---------|------|
| 發瘋（`madness`） | 黃色符號、瘋狂耳語 — 核心手段 | 極高 |
| 弱化（`weakened`） | 精神動搖導致行動猶豫 | 高 |
| 沈默（`silence`） | 封鎖施法者（黃衣之王的壓制） | 中 |
| 黑暗（`darkness`） | 精神迷霧，看不清現實 | 中 |
| 標記（`marked`） | 黃色印記標記目標，增加後續傷害 | 低（僅黃衣之王） |

### 使用的正面狀態（怪物自身）

| 狀態 | 使用場景 | 頻率 |
|------|---------|------|
| 護盾（`ward`） | 精神護壁，降低來自調查員的恐懼反擊 | 高 |
| 隱蔽（`stealth`） | 幻影般的存在，難以鎖定 | 中 |

### 狀態免疫

| 狀態 | 原因 |
|------|------|
| 發瘋（`madness`） | 本身就是瘋狂的化身 |
| 黑暗（`darkness`） | 在黑暗中如魚得水 |

### 位階範圍

| 怪物 | 位階 |
|------|------|
| 黃衣信徒（邪教徒變體） | 雜兵 — 威脅 |
| 拜亞基 | 威脅 — 精英 |
| 羅伊格爾與札爾 | 精英 — 頭目 |
| **黃衣之王** | **頭目** |
| 伊塔庫亞 | 頭目 — 巨頭 |
| 哈斯塔 | 巨頭（不可擊敗） |

### 恐懼特色

| 恐懼半徑 | 恐懼值 | 設計意圖 |
|---------|--------|---------|
| 3–5（遠） | 1–2（中低） | 瀰漫性，遠遠就開始侵蝕，持續累積而非一擊崩潰 |

### 敵對家族

克蘇魯眷族（風 vs 水）
